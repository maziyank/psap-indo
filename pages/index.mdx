# Standar Akuntansi Pemerintahan (SAP)
import { Callout } from 'nextra/components'

## Tentang

Website ini adalah sebuah upaya menyasikan Standar Akuntansi Pemerintah (SAP) dengan cara penyajian yang lebih menarik dan memudahkan pengguna. Disini pengguna dapat dengan mudah membaca isi Standar karena adanya navigasi dan fitur pencarian.

## Berkontribusi

Proyek ini adalah proyek terbuka. Jika anda aingin berkontribusi, silahkan membuat issue atau Pull Request langsung pada repository [github ini.](https://github.com/maziyank/psap-indo)

## Disclaimer

<Callout type="warning">
  Semua konten website ini dikonversi secara semi-otomatis dari [Buku SAP 2021](https://www.ksap.org/sap/wp-content/uploads/2021/11/Buku-SAP-2021.pdf) yang ditetapkan melalui [Peraturan Pemerintah Nomor 71 Tahun 2010](https://www.ksap.org/sap/wp-content/uploads/2012/08/PP_71_TAHUN_2010-_-oke.pdf). Sehingga apabila terdapat perbedaan, pengguna harus merujuk pada dokumen sumber asli.
</Callout>
